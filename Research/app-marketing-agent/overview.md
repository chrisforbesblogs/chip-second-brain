# App Marketing Agent

Research and planning for an AI-assisted marketing agent setup for our apps.

## Goal

Design an agent/tooling workflow for marketing apps using modern AI techniques, social media content creation, campaign planning, analytics, and channel-specific publishing support.

## Storage

- Research notes live in `second-brain/Research/app-marketing-agent/`.
- Use this folder for marketing-agent research, campaign workflows, channel analysis, and draft operating models.

## Suggested Sections

- `product/` - target use cases, personas, workflows, roadmap
- `research/` - AI marketing techniques, social media tooling, platform research
- `design/` - workflow diagrams, UI concepts, dashboard mockups
- `features/` - backlog and specs
- `implementation/` - architecture, integrations, data model, build plans
- `testing/` - test plans and QA notes
- `decisions/` - dated decision records
