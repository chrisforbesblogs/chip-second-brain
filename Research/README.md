# Research

Use this folder for exploratory research and marketing-related captures that should live in the second brain rather than the second-brain project structure.

Default here:
- Marketing requests
- Audience and channel research
- Campaign ideas
- Social/content drafts
- Competitor and positioning notes
- Light app-market validation notes

Use `../Projects` only when Chris explicitly asks for a formal project document such as a brief, feature spec, implementation plan, decision record, release note, or durable project report.

When moving or summarizing material from a Git-backed project folder, do not copy `.git` directories, remote URLs, account details, credentials, or repository metadata into this folder.
