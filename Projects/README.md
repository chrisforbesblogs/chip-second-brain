Central planning and collateral folder for active projects inside the second brain.

This folder is for non-source project material: briefs, roadmaps, research, designs, feature specs, implementation plans, decisions, release notes, and links to source repositories.

Buildable application source code lives separately under:

```text

```

## Rules

- Keep source code out of `second-brain/Projects`.
- Keep generated folders such as `node_modules`, `.expo`, `dist`, and `build` out of project notes.
- Put each buildable project in its own Git repo under `/home/cef-admin/projects-source-code`.
- Link each project folder here to its source repo and local source path.
