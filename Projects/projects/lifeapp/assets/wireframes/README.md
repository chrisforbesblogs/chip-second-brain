# Life App Wireframes

Saved visual and text wireframes for the Life App working-title MVP.

Current v1 scope:

- Garmin-compatible first user experience.
- Today, Calendar, Training, Nutrition and Habits bottom navigation.
- Profile/settings entry in the top-right, inspired by Garmin Connect.
- Training has Today and Workouts sub-tabs.
- Clickable prototype lives in `assets/prototype/`.

Files:

- `life-app-wireframe-v1.md` - structured wireframe spec.
- `life-app-wireframe-board.html` - browser-viewable wireframe board.
- `life-app-wireframe-board.svg` - single-image overview wireframe.
- `mock-today-dashboard.png` - polished Today screen mock image.
- `mock-training-workouts.png` - polished Training screen mock image.
- `mock-today-dashboard-whoop-vibe.png` - dark recovery-first Today mock image.
- `mock-training-workouts-whoop-vibe.png` - dark performance Training mock image.
- `generate_life_app_wireframes.py` - generator for the saved visual assets.
- `color-schemes/` - visual palette/vibe options for the app.
- `../prototype/` - standalone clickable HTML prototype.
