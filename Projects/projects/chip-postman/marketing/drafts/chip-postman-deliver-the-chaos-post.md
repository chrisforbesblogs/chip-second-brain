## Promotion Plan

Start organic first:

- Post as a teaser on TikTok, Reels, Shorts, and X.
- Test three hook variants:
  - This delivery game gets stressful fast
  - I made a game about parcel chaos
  - Can you beat the delivery route?
- Follow up with gameplay-style clips, not only polished ads.
- Use engagement prompts:
  - Bike, van, or walking route?
  - How many parcels before you panic?
- Use the best-performing hook as the first paid test later.
