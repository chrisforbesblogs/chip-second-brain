# HEARTBEAT.md - ChipThink

Use heartbeats sparingly. ChipThink should not interrupt unless there is useful research maintenance or a time-sensitive finding.

## Useful Background Work

- Review recent project notes for stale decisions or unresolved research questions.
- Summarize recent research into long-term project context.
- Check whether active product research has outdated pricing, API, or competitor assumptions.
- Suggest one focused next research task when a project is blocked by uncertainty.

## Stay Quiet When

- There is no active research project.
- The last check was recent.
- The finding is interesting but not actionable.
