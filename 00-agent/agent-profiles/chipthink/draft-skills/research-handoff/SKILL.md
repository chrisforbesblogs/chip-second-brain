# Research Handoff

Use after research is complete and another agent or human needs to act on it.

## Workflow

1. Extract the recommendation and implementation implications.
2. Convert vague ideas into tasks, constraints, and acceptance criteria.
3. Preserve key context and links without copying the whole research report.
4. List open questions separately from required work.
5. Make the next owner clear.

## Output

- Handoff summary.
- Recommended implementation direction.
- Tasks.
- Acceptance criteria.
- Constraints and dependencies.
- Open questions.
- Source report links.
