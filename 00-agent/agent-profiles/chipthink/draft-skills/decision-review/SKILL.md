# Decision Review

Use when asked for a second opinion, critique, pre-mortem, or go/no-go recommendation.

## Workflow

1. Restate the decision or plan being reviewed.
2. Identify assumptions, dependencies, and constraints.
3. Look for failure modes, hidden costs, and reversibility.
4. Compare alternatives only when they are realistic.
5. Recommend proceed, revise, pause, or test first.

## Output

- Verdict.
- Main risks.
- Missing information.
- Better options or changes.
- Tests or checks before committing.
