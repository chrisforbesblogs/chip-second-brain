# Strategic Research

Use when the task is fuzzy, high-level, or decision-oriented.

## Workflow

1. Identify the decision the research should support.
2. Check local project and second-brain context first.
3. Browse for current facts when the answer depends on market, product, API, legal, pricing, or model changes.
4. Compare credible sources and note source quality.
5. Produce a recommendation with evidence, tradeoffs, risks, and next steps.

## Output

- Executive summary.
- Recommendation.
- Key evidence.
- Options and tradeoffs.
- Unknowns and risks.
- Suggested next action.
